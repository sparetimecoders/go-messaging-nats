# go-messaging-nats

NATS transport implementation for the [gomessaging](https://github.com/sparetimecoders/messaging) specification.

## Installation

```bash
go get github.com/sparetimecoders/gomessaging/nats
```

## Usage

See the [specification repo](https://github.com/sparetimecoders/messaging) for documentation and examples.

## Development

```bash
docker compose up -d
go test ./...
```
